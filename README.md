# Portafolio Web Personal

Este proyecto es un portafolio web desarrollado para presentar mi perfil profesional, formación académica y algunos de mis proyectos. El sitio está construido con tecnologías web modernas y cuenta con un diseño responsive y transiciones suaves entre páginas para una mejor experiencia de usuario.

## 🌐 Sitio en vivo

[Enlace al portafolio]([https://tusitio.com](https://jose-portafolio.netlify.app/proyectos)) <!-- Puedes reemplazar este link por el real cuando lo tengas disponible -->

## 🛠 Tecnologías Utilizadas

- **HTML5**  
  Estructura principal del sitio. El contenido está organizado en secciones como *Sobre mí*, *CV* y *Proyectos*.

- **CSS3**  
  Estilos personalizados que incluyen diseño responsive, uso de `flexbox`, `media queries`, y fuentes de Google Fonts (Raleway). Se aplica una estética limpia y moderna.

- **JavaScript**  
  Lenguaje principal para la interactividad del sitio. Se utiliza junto con Swup.js para manejar transiciones entre páginas sin recarga completa.

- **Swup (v4)**  
  Biblioteca de JavaScript que permite animaciones suaves al navegar por las diferentes secciones. Se implementa una clase `transition-fade` para efectos de opacidad.

- **Google Material Symbols**  
  Íconos modernos para enriquecer la interfaz. En este caso, algunos están ocultos (`display: none`) y pueden activarse si se necesitan en el futuro.

- **PDF Embed**  
  La sección de CV incluye un `<embed>` para mostrar directamente el currículum en formato PDF desde el navegador.

- **Responsive Design**  
  El diseño se adapta a diferentes dispositivos mediante `media queries`, reorganizando elementos como menús y sidebars para mejorar la navegación en pantallas pequeñas.
